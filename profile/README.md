# CAPIDATA

> Comunidade Brasileira de Arquitetura, Processamento e Inteligência em Dados

A CAPIDATA é uma comunidade open-source e sem fins lucrativos, dedicada a construir soluções em processamento, governança e inteligência em dados, com identidade 100% brasileira.

Inspirada no Rio Capibaribe e no mestre do frevo Capiba, nossa comunidade tece arquitetura e código para que dados fluam com qualidade, transparência e segurança de ponta a ponta.
