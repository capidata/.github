# Membros da CAPIDATA

Registro público dos membros da comunidade por nível de participação.

> *"A participação na CAPIDATA organiza-se em quatro níveis não hierárquicos,*
> *cada qual com direitos e responsabilidades específicos."*
> — Pacto Fundante, Art. 5°

---

## Sobre este registro

Este arquivo é o registro público de quem compõe a CAPIDATA.

Ele é **autogerido**: cada pessoa adiciona seu nome ao nível correspondente via Pull Request — que é, ao mesmo tempo, sua adesão expressa ao [Pacto Fundante](../estatuto/pacto-fundante-v0.1.md). O histórico de commits do Git é o registro imutável de quando cada pessoa entrou.

## 🌟 Membros Plenos

Membros com direito a voto em todas as decisões e elegíveis para cargos de governança.

Para se tornar Membro Pleno:
1. Leia e adira ao [Pacto Fundante](../estatuto/pacto-fundante-v0.1.md)
2. Abra um PR adicionando seu nome a esta tabela
3. Inclua no PR: sua área de atuação, território e breve motivação

**Brasileiros e latino-americanos:** PR direto após contribuição verificável.

**Estrangeiros não latino-americanos:** consulte o processo de certificação no [Pacto Fundante, Art. 9°, inciso II](../estatuto/pacto-fundante-v0.1.md#nível-iv--membros-plenos).

## Como assinar o Pacto Fundante

Adicionar seu nome à tabela de Membros Plenos via PR é o ato de assinatura do Pacto. Simples, rastreável, sem cartório.

```bash
# Fork o repositório .github
# Edite este arquivo adicionando sua linha na tabela
# Abra um PR com o título: [MEMBRO] Seu Nome

# O PR deve conter:
# - Sua linha na tabela de Membros Plenos
# - Uma linha no commit: "Adiro ao Pacto Fundante da CAPIDATA v0.1"
```
